Atos Property Analytics SCS

Atos Property Analytics solution provides a reporting solution for the Oracle Property Manager modules within Oracle E-Business Suite. It builds on Oracle’s BI technology and comes with pre-build table structure; Extract-Transform-Load (ETL) routines and a list of reports/dashboards, which can be easily integrated with an existing Oracle BI Application deployment.

Features

  • Highly interactive, role-based reports and dashboards
  • Pre-build table structure from Property modules of e-Business suite application
  • Pre-build ETL routines to load data into data warehouse tables
  • Pre-built semantic model with entity relationship based on best practices

Benefits

  • Pre-build metrics, reports and dashboard
  • Value to the customer from day 1 of its implementation
  • Deployed as is or modified to fit customer business requirements
  • Semantic model with relationships & metrics to cater reporting requirements

Pricing

  • Including VAT
  • Minimum contract period: Day
